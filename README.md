# Finbuddy: 자산 관리 플랫폼

### 프로젝트 기간
2025.02.10 ~ 2025.03.12

### 프로젝트 개요
모든 계좌를 한 곳에서 조회하여 효율적으로 지출을 관리하고, 안전한 송금 및 자동 이체 등 안전한 금융 서비스를 제공합니다.

### 주요 기능
- 마이데이터 연동 (더미데이터 생성)
- 거래 내역 분석 및 시각화
- 이체 및 자동 이체
- 예산 관리 및 알림
- 금융 상품 추천 및 조회
